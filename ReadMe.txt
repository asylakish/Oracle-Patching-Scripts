Скрипт для параллельной установки патчей Oracle. 
В скрипте RuInstaller.sh можно ограничить количество одновременно работающих заданий с помощью параметра parallel_limit.
Также в скрипте RuInstaller.sh прописывается переменная $ORACLE_HOME.

Скрипт принимает 2 параметра:
1й - это файл с именами БД на которые нужно установить патчи,
2й - это скрипт с командами которые нужно выполнить.

Пример запуска:
./RuInstaller.sh SidList.txt datapatch_verbose.sh
